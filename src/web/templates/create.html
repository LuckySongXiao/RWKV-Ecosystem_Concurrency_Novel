📖 什么是 SKILL.md？
SKILL.md 是描述写作风格技能的 Markdown 文件，遵循 Anthropic 规范。 写作时会将所有已激活的 SKILL 自动注入到 Prompt，约束 AI 模仿该文风。
格式：顶部 YAML 描述（name / description），下方为写作风格正文。
