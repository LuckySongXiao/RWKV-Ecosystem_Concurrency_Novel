# 世界观设定

## 题材
仙侠、道士、僵尸

## 世界观
天地灵气循环，三界（天界/人界/冥界）各有法则。天道运转以劫数为纪，万载一轮回。灵气浓度决定修行上限，天地大劫时灵气枯竭。

## 修行体系
境界划分：
1. 炼气期（1-9层）→ 感气、引气、御气
2. 筑基期（初/中/后/大圆满）→ 凝聚道基
3. 金丹期 → 结丹成婴之前提
4. 元婴期 → 神魂出窍，可夺舍
5. 化虚期 → 空间法则初窥
6. 合体期 → 天人合一
7. 大乘期 → 渡劫飞升之前
8. 渡劫期 → 天劫洗礼
突破条件：需灵气积累+悟道契机+天劫考验（金丹以上）

## 主要人物
**凌云** (主角)
  身份: 隐匿的青云宗外门弟子/天命之子/上古仙人后裔
  性格: 坚韧不拔、重情重义、心怀正道、极具危机意识、善于借势
  实力: 炼气期（9）→ 筑基初期（小圆满）
  背景: 本为青云宗外门灵根残缺、备受冷眼的废柴弟子，实为上古混沌圣女转世，被宗门大能用禁术封印灵根并伪装成废物以躲避天劫。在一次宗门禁地秘境的混沌道韵逆冲中，封印破碎，展现出‘混沌道体’的恢弘潜力，成为万年难遇的修行奇才。在探索秘境时，意外获得上古仙人‘九转轮回图’的传承，肩负起稳固三界灵脉、阻止‘蚀灵大劫’的重任。

**墨渊** (重要配角/反派转正)
  身份: 血煞殿太子/血殿隐脉继承人/魔道天骄
  性格: 冷静深沉、野心勃勃、坚守己道、善用谋略、亦正亦邪
  实力: 金丹后期 → 元婴初期（大圆满）
  背景: 血煞殿继承人，是上古魔尊与人界王后私生子，因半妖之身和野心，被正道围剿家族时侥幸逃脱。他弑父，夺权，以血祭仪式窃取‘九幽恶血’成道，魔威盖世。因血脉之险，其体内魔气与生灵之气逐渐形成冲突，必须找到平衡之道。在目睹正魔双方对抗中如何导致修行崩坏后，内心对‘究竟之道’产生怀疑，决定走一条超越善恶的独特之路。

**素灵** (导师/阵营核心支柱)
  身份: 青云宗首席长老/太上医修圣女/前仙庭药师
  性格: 仁慈宽厚、智慧绝顶、顾全大局、明察秋毫、历经沧桑
  实力: 渡劫期 → 飞升失败陨落
  背景: 当年天界最负盛名的仙药圣女，因不忍见凡人疾苦，自毁仙体下界创立青云宗医修堂，炼丹医道双绝。万年前圣器封印松动，灵气沉降至冰点，她以毕生修为为保阵列的唯一活口。如今灵气复苏，却因误入‘九幽噬魂诀’的引导，她的肉身灵魂已陷入深度沉眠，只能将部分意识封印在宗门内代代相传的‘灵韵丹炉’中，继续研究延长宗门弟子寿命的法门。

**幽岚** (重要配角/反派)
  身份: 天庭巡查使/太上仙
  性格: 严肃威严、铁面无私、恪守天规、冷静高冷、神秘莫测
  实力: 渡劫期 → 合体期大圆满
  背景: 天庭‘巡天司’的首席天官，拥有无上仙源，但其身份是在三界之战中神秘失踪的仙祖后人。为追寻流落三界的‘九彩仙玉’（传说中可修复天裂并消除所有道种引发的劫数的灵器）而下凡，化身‘幽岚’。他对三界秩序的维护极其偏执，会毫不留情地清除所有‘劫数源头’，包括凌云身具的‘混沌灵脉’与墨渊的‘血煞魔血’。

**幽冥** (重要配角/补足善恶平衡)
  身份: 幽冥宗少宗主/秘术宗门
  性格: 神秘莫测、精明狡猾、言语简洁、极度敏锐、对至高无上的力量有狂热追求
  实力: 金丹初期 → 结丹中期
  背景: 幽冥宗为三界曾经的宗门，掌管开创了“幽冥秘术”，传承有大威力神通《幽冥噬魂诀》，专攻灵体和神魂的领域。宗门本身在“蚀灵大劫”中覆灭，只留下幽冥圣灵一脉。他曾联手墨渊行事，最终因目睹天地之劫力量的真相而醒悟，决定用一切代价去将‘蚀灵大劫’的传承从三界抹去，避免灾难重演。

**白羽** (重要配角/正面伙伴)
  身份: 青云宗弟子/冰系剑修/慕容族长子女
  性格: 冷静自持、故作神秘、坚定果断、温情似火、智商奇高
  实力: 筑基期 → 金丹期（初期）
  背景: 青云宗元婴期实力宗主之女，天生与古冰魄道具有强大的亲和，元神包裹着一粒极寒的晶石，令她修习剑术达到峰值，为剑胚。因家族为法则瓦解而被灭族，幸得一位年长灵体带入人间秘境，从此在家族废墟之中孤独修行。她追逐着‘道源’的传说，希望重建青云宗，于万劫之中带领正道开启新篇。


## 势力格局
宗门（正道/魔道/散修）、世家大族、天庭/冥府、妖族势力

## 经济体系
灵石（下/中/上/极品）为通用货币，丹药/法器/灵材为交易核心，宗门掌控灵矿资源

## 核心冲突类型
道争（正魔之争）、资源争夺（灵矿/秘境）、天劫危机、上古秘辛、种族矛盾（人妖）

## 力量上限
渡劫期飞升为上限，天界仙人不可轻易下界

## 故事主线
**核心冲突**: 当三界稳定的天地灵气渐趋枯竭，导致下一个毁灭性大劫迫近，凌云这位拥有混沌道体的凡人天才，必须抵御内心魔性、团结各方反对者，在风雨飘摇的三界中探寻拯救众生的唯一希望，并对抗意图利用劫数完成魔道伟业的宿敌墨渊。

**故事弧线**:
  起: 【起】天命与残缺：灵气枯竭预兆显现，宗门内因灵根资质的审判而暗流涌动。外门弟子凌云在一次绝境中的生死抉择（可能是为救人而触碰禁忌法坛），觉醒混沌道体，内在的纯善与道义之心驱使他背负起寻找救赎三界之法的使命，并与宗门初次产生理念的隔阂。
  承: 【承】师者之殒与道路抉择：天庭降下针对“异类”（仙人/妖族/魔修）的天罚征兆，素问仙子为保护宗门安全和凌云被迫与天庭产生冲突，最终为掩护众人而陨落。墨渊因父亲之死而对大劫产生怀疑，开始秘密与凌云接触，凌云在悲痛和质疑中，融合了正道与魔道的修行法门，力量飞跃却也因此引来更大的追杀与内心挣扎。
  转: 【转】黑暗的短暂光明：凌云等人揭示出大劫背后是上界天庭对下界修行者的清理计划，为维持自身统治，天庭（或某些长老）有意引导或加剧天地灵气枯竭。凌云、墨渊、素问的残魂指引下，组建了一个超越正魔界限的实验性联盟，计划是寻找上古（混沌仙祖）遗产中的虚空灵源以重塑灵气循环体系。然而计划被叛徒和更强大的势力（如守护传统修行体系的部分仙君、试图垄断灵脉的魔修大佬）阻挠，联盟濒临毁灭。
  合: 【合】道心与劫数的终局：为对抗末日，凌云必须承担混沌道体的全部风险，选择一条未知的道路。最终在决战中，灵气回归的临界点上，混沌道体的代价显现（身死/道果苍化/维持虚空灵源而自身永为凡体）。墨渊在大劫中领悟生生不息之道，成为新一代魔道守护者。而凌云的故事和抉择，化为新时代的传说与指引，开启了后三界统一的新纪元。
