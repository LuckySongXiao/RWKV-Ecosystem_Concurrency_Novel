# 世界观设定

## 题材
仙侠、道士、僵尸

## 世界观
天地灵气循环，三界（天界/人界/冥界）各有法则。天道运转以劫数为纪，万载一轮回。灵气浓度决定修行上限，天地大劫时灵气枯竭。

## 修行体系
境界划分：
1. 炼气期（1-9层）→ 感气、引气、御气
2. 筑基期（初/中/后/大圆满）→ 凝聚道基
3. 金丹期 → 结丹成婴之前提
4. 元婴期 → 神魂出窍，可夺舍
5. 化虚期 → 空间法则初窥
6. 合体期 → 天人合一
7. 大乘期 → 渡劫飞升之前
8. 渡劫期 → 天劫洗礼
突破条件：需灵气积累+悟道契机+天劫考验（金丹以上）

## 主要人物
**凌云** (主角)
  身份: 青云宗外门弟子/天命之子
  性格: 坚韧不拔、重情义、心怀正道
  实力: 炼气期巅峰(9层)→ 筑基初期(小圆满)
  背景: 出身寒微,因灵根残缺被宗门抛弃。在一次秘境探险中意外获得上古仙人传承,觉醒'混沌道体',成为万年难遇的修行奇才。背负着拯救三界于劫数的使命。

**墨渊** (重要配角/反派转正道伙伴)
  身份: 幽冥殿少主/魔道天骄
  性格: 冷酷无情、野心勃勃、深藏不露
  实力: 金丹后期 → 元婴初期(大圆满)
  背景: 魔尊之子,自幼被培养为继承人。因目睹父亲被正道围攻而对正邪之分产生质疑,暗中与凌云结盟,试图打破三界僵局。掌握禁忌魔功《九幽噬魂诀》。

**素问仙子** (导师/阵营核心支柱)
  身份: 青云宗首席长老/医修圣手
  性格: 温婉慈悲、智慧通达
  实力: 化虚期 → 合体期(大圆满)→ 渡劫失败陨落
  背景: 曾是天庭药王谷弟子,因不忍见凡人受苦下山创立青云宗。精通炼丹术与治疗法术,拥有'九转还魂草'等神物。在劫数中牺牲自己保护凌云突破瓶颈。

## 势力格局
宗门（正道/魔道/散修）、世家大族、天庭/冥府、妖族势力

## 经济体系
灵石（下/中/上/极品）为通用货币，丹药/法器/灵材为交易核心，宗门掌控灵矿资源

## 核心冲突类型
道争（正魔之争）、资源争夺（灵矿/秘境）、天劫危机、上古秘辛、种族矛盾（人妖）

## 力量上限
渡劫期飞升为上限，天界仙人不可轻易下界

## 故事主线
**核心冲突**: 正道魁首‘天枢宗’为应对万载一轮的天地大劫，启动禁忌秘法‘九转归元阵’强行抽取灵脉延续文明，却意外撕裂三界屏障，导致冥府僵尸军团入侵人间。主角作为唯一能同时驾驭仙、道、魔之力的异数，必须在拯救苍生与维护三界平衡间做出抉择。

**故事弧线**:
  起: 散修少年林凡因体内混沌灵根被迫卷入宗门纷争，在逃亡中觉醒上古血脉传承‘太虚神瞳’，发现自身竟是封印冥府通道的关键钥匙。
  承: 联合妖族遗孤与叛逃天庭剑仙组建‘破妄盟’，以非正统手段对抗僵尸潮与正道围剿。在夺取上古秘境‘归墟之眼’时揭开世家大族操纵灵矿资源的阴谋。
  转: 为阻止僵尸王吞噬灵脉核心，林凡被迫吸收禁忌魔功《九幽玄典》获得力量反噬，肉身崩毁仅存元婴状态。此举引发天劫提前降临并撕裂空间通道。
  合: 以残魂为祭献启动逆转阵法，将三界屏障重构为新型平衡体系。最终化作守护三界的‘混沌碑’永恒镇守归墟之眼，但留下可能复苏的太虚神瞳血脉伏笔。
